Enable access to local files

Text Grab needs one permission to read PDFs and pages opened from your computer

You started Text Grab on a file:// page (a PDF or web page opened from your computer). Chrome blocks extensions from reading local files unless you turn on a per-extension permission — so nothing happened.

To allow it:

  1. Open the extension's details page (the button below opens it).
  2. Turn on Allow access to file URLs.
  3. Return to your local PDF or page and start Text Grab again.
